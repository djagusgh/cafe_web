## cafe_web

- 프로젝트 개요
  1) flask 과 mariaDB와의 연동을 통해 DB의 내용을 웹사이트에 표시
  2) Session을 활용한 로그인 상태 유지 및 로그아웃 기능 구현
  3) 각 카페별 후기 추가 기능 구현
